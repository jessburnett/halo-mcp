Halo-MCP is a standalone research environment designed for quick-setup compliance in chatbots and AI agents. It prioritizes **Zero-Knowledge Proofs (ZKP)** and **Local Semantic Redaction**.

## 🔬 Verified Capabilities
As of **April 27, 2026**, this project successfully demonstrates:
- **Contextual Redaction**: Using a 1.5B parameter SLM to identify PII through sentence structure rather than simple regex.
- **AgeKey Handshake**: Supporting the **OpenAge Initiative** standard for cryptographic age estimation without identity storage.
